## 1. Vectores Aleatorios y Función de Probabilidad Conjunta

Un **vector aleatorio** bivariado $(X,Y)$ es una función matemática que asigna un vector numérico en $\mathbb{R}^2$ a cada elemento $\omega$ del espacio muestral $\Omega$. Es decir, se registran conjuntamente dos mediciones o características del mismo experimento aleatorio.

### Caso Discreto

Para variables aleatorias discretas, la **función de probabilidad conjunta** se define como $f(x,y) = P(X=x, Y=y)$, la cual describe la probabilidad de que $X$ tome el valor $x$ y simultáneamente $Y$ tome el valor $y$.

Debe cumplir dos propiedades fundamentales:
1. $f(x,y) \ge 0$
2. $\sum_x \sum_y f(x,y) = 1$

> [!info] Cálculo de probabilidades (Discreto)
> Para cualquier región $A \subset \mathbb{R}^2$, la probabilidad se calcula sumando en los puntos de la región: $P((X,Y) \in A) = \sum\sum_{(x,y) \in A} f(x,y)$.

### Caso Continuo

Para variables aleatorias continuas, la **función de densidad conjunta** $f(x,y)$ es una superficie sobre el plano $\mathbb{R}^2$ (siempre $f(x,y) \ge 0$) cuyo volumen total subyacente es exactamente igual a 1.

$$\int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f(x,y) \,dx\,dy = 1$$

La probabilidad de que el vector tome valores en un rectángulo $[a,b] \times [c,d]$ se calcula mediante la doble integral sobre esa región:

$$P(a < X < b, c < Y < d) = \int_a^b \int_c^d f(x,y) \,dy\,dx$$

## 2. Distribuciones Marginales

La **función de probabilidad o densidad marginal** nos permite aislar y conocer la distribución individual de una de las variables (por ejemplo, $X$), "escondiendo" o "promediando" el comportamiento de la otra variable ($Y$). Se obtiene a partir de la [[#1. Vectores Aleatorios y Función de Probabilidad Conjunta|función de probabilidad conjunta]].

- **Para vectores continuos:** Se integra respecto a la variable que se desea eliminar.

$$f_X(x) = \int_{-\infty}^{\infty} f(x,y) \,dy \quad \text{y} \quad f_Y(y) = \int_{-\infty}^{\infty} f(x,y) \,dx$$

- **Para vectores discretos:** Se suma a lo largo de todos los valores posibles de la otra variable.

$$f_X(x) = \sum_y f(x,y) \quad \text{y} \quad f_Y(y) = \sum_x f(x,y)$$

> [!warning] Observación Teórica del Libro de Rincón
> Es importante destacar que **distintas funciones de densidad conjuntas pueden producir exactamente las mismas funciones marginales**. Por ende, tener solo las distribuciones individuales (marginales) no es suficiente para reconstruir cómo se relacionan conjuntamente, a menos que sean [[11. Distribuciones Bivariadas - Dist. Condicionales e Independencia#2. Independencia de Variables Aleatorias|independientes]].

## 3. Valor Esperado en Vectores Aleatorios

El **Valor Esperado** generalizado para funciones de dos variables $g(X,Y)$ se calcula de manera análoga a la esperanza de una sola variable, pero utilizando la [[#1. Vectores Aleatorios y Función de Probabilidad Conjunta|distribución conjunta]]:

- **Continuo:** $E[g(X,Y)] = \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} g(x,y)f(x,y) \,dx\,dy$
- **Discreto:** $E[g(X,Y)] = \sum_x \sum_y g(x,y)f(x,y)$

*(Nota: Para encontrar la esperanza individual $E(X)$, se puede usar $g(X,Y) = x$ dentro de la fórmula conjunta o simplemente usar la [[#2. Distribuciones Marginales|distribución marginal]] $f_X(x)$).*

## 4. Varianza y Covarianza

La **Varianza** individual de cada variable aleatoria en el vector mantiene su definición habitual para medir su propia dispersión:

$$V(X) = E(X^2) - [E(X)]^2$$

donde $E(X^2)$ se obtiene usando el [[#3. Valor Esperado en Vectores Aleatorios|Valor Esperado generalizado]].

La **Covarianza** nace al buscar cómo varía una variable respecto a la otra; cuantifica el grado de **asociación lineal** entre $X$ e $Y$.

Se define como:

$$Cov(X,Y) = E[(X - E(X))(Y - E(Y))]$$

Para propósitos de cálculo se suele usar la fórmula reducida:

$$Cov(X,Y) = E(XY) - E(X)E(Y)$$

donde $E(XY)$ es el [[#3. Valor Esperado en Vectores Aleatorios|valor esperado]] usando $g(X,Y)=xy$.

### Propiedades de la Covarianza:

1. $Cov(X,a) = 0$ (siendo $a$ constante).
2. $Cov(X,X) = V(X)$ (la covarianza de una variable consigo misma es su varianza).
3. **Simetría:** $Cov(X,Y) = Cov(Y,X)$.
4. **Varianza de la Suma:** $V(X \pm Y) = V(X) + V(Y) \pm 2\,Cov(X,Y)$ — ver extensión bajo [[11. Distribuciones Bivariadas - Dist. Condicionales e Independencia#Propiedades clave bajo independencia|independencia en Semana 11]].

> [!tip] Relación entre Covarianza e Independencia
> Según detalla Rincón, **si $X$ e $Y$ son independientes, entonces obligatoriamente $Cov(X,Y) = 0$**.
> Sin embargo, el inverso no es cierto: **el hecho de que $Cov(X,Y) = 0$ NO implica que las variables sean independientes**, ya que podrían tener una relación de tipo no lineal que la [[#5. Correlación (ρ)|correlación]] tampoco puede detectar. La independencia se formaliza en la [[11. Distribuciones Bivariadas - Dist. Condicionales e Independencia#2. Independencia de Variables Aleatorias|Semana 11]].

## 5. Correlación (ρ)

La [[#4. Varianza y Covarianza|covarianza]] depende de las unidades de medida de las variables. Para obtener una métrica estandarizada que mida exclusivamente el **grado de dependencia lineal**, se utiliza el **Coeficiente de Correlación** ($\rho_{XY}$).

$$\rho_{XY} = \frac{Cov(X,Y)}{\sigma_X \sigma_Y} = \frac{Cov(X,Y)}{\sqrt{V(X)\,V(Y)}}$$

donde $\sigma_X = \sqrt{V(X)}$ y $\sigma_Y = \sqrt{V(Y)}$ son las desviaciones estándar individuales obtenidas de las [[#2. Distribuciones Marginales|distribuciones marginales]].

### Interpretación del Coeficiente de Correlación:

El coeficiente de correlación tiene como dominio estricto el intervalo $[-1, 1]$.

- **Si $\rho_{XY} = 1$:** Existe una asociación lineal directa perfecta (a mayor $X$, mayor $Y$ estrictamente sobre una recta ascendente).
- **Si $\rho_{XY} = -1$:** Existe una asociación lineal inversa perfecta (a mayor $X$, menor $Y$ sobre una recta descendente).
- **Si $\rho_{XY} = 0$:** No existe asociación lineal entre las variables (aunque podría existir otro tipo de dependencia no capturada por la [[#4. Varianza y Covarianza|covarianza]]).
- *Convención general:* Valores $|\rho| \ge 0.7$ suelen indicar una correlación "fuerte".
